# Project Architecture

This is a Next.js application based on Feature-Sliced-Design architecture.

- Sources in the project are located in `src` folder.
- All pages are located in `src/pages/`.
- All features are located in `src/features/`.
- All widgets are located in `src/widgets/`.
- All shared components are located in `src/shared/`.
- Shadcn ui kit located in `src/shared/ui/shadcn/`.
- App shell located in `src/app/`.

## Coding Standards

- Use TypeScript for all new files
- Follow the existing naming conventions
